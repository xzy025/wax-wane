# 老师交易方法论知识库

在此目录存放老师的交易方法论文稿。

## 目录结构

```
teacher/
├── README.md           # 本文件
├── principles/         # 核心理念
│   └── *.md           # 每个文件一个主题
├── patterns/           # 交易模式
│   └── *.md           # 买点、卖点特征
├── cases/              # 案例库
│   └── *.md           # 成功/失败案例
├── risk-rules/         # 风控规则
│   └── *.md           # 仓位、止损、止盈规则
└── raw/                # 原始文稿
    └── *.md           # 未整理的原始内容
```

## 文件格式建议

每个 `.md` 文件建议包含：

```markdown
# 标题

## 要点
- 要点1
- 要点2

## 详细说明
...

## 案例（如有）
...
```

## 示例

### principles/core.md
```markdown
# 核心交易理念

## 只做主升浪
- 不参与震荡行情
- 不抄底，只做确认后的突破

## 龙头战法
- 只做板块龙头
- 不做跟风股
- 龙头见顶后空仓等待

## 严格止损
- 跌破5日均线立即止损
- 不补仓，不扛单
```

### patterns/buy-points.md
```markdown
# 买点特征

## 突破回踩
- 突破前高后回踩不破
- 回踩时成交量萎缩
- 回踩到支撑位时入场

## 首板放量
- 首板涨停，成交量放大
- 板块有3只以上涨停
- 大盘处于上升趋势
```

### cases/success-001.md
```markdown
# 案例：XXX股票

## 买入
- 日期：2026-01-15
- 理由：突破前高，板块龙头，大盘配合
- 仓位：30%

## 持有
- 3天涨20%
- 期间加仓至60%

## 卖出
- 日期：2026-01-18
- 理由：放量滞涨，板块分化
- 盈利：+18%

## 教训
- 应该更早止盈
- 板块分化时要及时离场
```
