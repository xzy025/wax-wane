# 云聪老师核心交易理念

## 交易风格
- 偏好反转交易（2B 模型）
- 注重位置和结构
- 分批分仓，控制风险
- 结合资金流和板块热点

## 核心原则

### 1. 位置优先
- 必须到达重要支撑/压力位才考虑入场
- 不追高，不抄底（等待结构完成）
- 大形态边缘不做（危险）

### 2. 结构确认
- 必须有明确的五浪模型
- 日线级别最可靠，60 分钟/15 分钟次之
- 结构不明显时降低仓位

### 3. 速度要求
- 2B 反转必须快速（第二天反弹）
- 拖延的反转不可靠
- 跌破后慢慢爬回来的不算 2B

### 4. 资金验证
- 必须结合资金流
- 没有资金流入的 2B 不做
- 资金驱动率要配合
- 量比 + 资金流入综合判断

### 5. 板块热点配合
- 当下热点板块优先
- 游资最看重的主线
- 非主线、非热点不做
- 板块有持续性

### 6. 分批操作
- 不要一次满仓
- 2B 买第一批，右肩加第二批
- 资金分成三份，分批建仓
- 保留资金应对意外

### 7. 容错设计
- 2B 失败不等于亏损
- 失败后走成头肩顶，继续上涨概率大
- 不要轻易止损，耐心持有
- 考虑被锤情况，设置好加仓位置

## 风控规则
- 2B 位置不要轻易止损
- 分批分仓降低风险
- 失败后等右肩位置加仓
- 再往下跌可以补缺口再买
- 大盘不好时轻仓或不做

## 三大交易模式

### 1. 2B 买入模型
- 支撑压力 + 五浪模型 + K 线组合
- 跌破后第二天快速拉起
- 失败后走头肩顶

### 2. 强势股低吸
- 强势股回调 50% 位置
- 必须有资金流入
- 大盘配合

### 3. 首板涨停模式
- 指数情绪 + 板块情绪 + 赚钱效应
- 按资金流 + 板块热点选股
- 大形态突破、茶杯带柄、突破回抽确认

## 适用场景
- 个股日线/60 分钟/15 分钟级别
- 需要配合资金流和板块热点
- 适合有一定经验的交易者
- 大盘好时重仓，大盘不好时轻仓或不做
