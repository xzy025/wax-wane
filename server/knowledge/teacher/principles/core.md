# 云聪老师核心交易理念

## 交易风格
- 偏好反转交易（2B 模型）
- 注重位置和结构
- 分批分仓，控制风险

## 核心原则

### 1. 位置优先
- 必须到达重要支撑/压力位才考虑入场
- 不追高，不抄底（等待结构完成）

### 2. 结构确认
- 必须有明确的五浪模型
- 日线级别最可靠，60 分钟/15 分钟次之
- 结构不明显时降低仓位

### 3. 速度要求
- 2B 反转必须快速（第二天反弹）
- 拖延的反转不可靠
- 跌破后慢慢爬回来的不算 2B

### 4. 资金验证
- 必须结合资金流
- 没有资金流入的 2B 不做
- 资金驱动率要配合

### 5. 分批操作
- 不要一次满仓
- 2B 买第一批，右肩加第二批
- 保留资金应对意外

### 6. 容错设计
- 2B 失败不等于亏损
- 失败后走成头肩顶，继续上涨概率大
- 不要轻易止损，耐心持有

## 风控规则
- 2B 位置不要轻易止损
- 分批分仓降低风险
- 失败后等右肩位置加仓
- 再往下跌可以补缺口再买

## 适用场景
- 个股日线/60 分钟/15 分钟级别
- 需要配合资金流和板块热点
- 适合有一定经验的交易者
