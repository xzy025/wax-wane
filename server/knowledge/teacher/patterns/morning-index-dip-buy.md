# A股早盘沪深300/中证500低吸策略

## 策略概述
- 等待指数跌到关键位置，低吸沪深300/中证500成分股
- 资金分成三份，分批建仓
- 结合资金流、板块热点、技术形态

## 操作步骤

### 第一步：建立股票池
- 沪深300成分股
- 中证500成分股
- 在软件中建立自选板块

### 第二步：等待指数跌到关键区间
- 支撑位（斐波那契、前期低点）
- **资金分成三份**：
  - 第一份：当前位置
  - 第二份：更低位置
  - 第三份：缺口位置

### 第三步：指数跌到位后，按资金流选股
- 板块热点优先
- 个股支撑压力位
- 日线/4小时/1小时买点

## 选股方法

### 方法一：资金流排行
- 使用同花顺等软件的资金流功能
- 选择主力资金流入的个股
- 结合市值判断（流入金额/市值比例）

### 方法二：量比 + 资金流
- 量比大的优先
- 右下角查看主力资金流入
- 量比大 + 资金流入 = 优选

### 方法三：行业板块
- 先看今天哪些板块在涨
- 只选上涨板块中的个股
- 有色金属、半导体、科技等

## 买入模型

### 双突破模型
```
分时图：
低点 → 次低点 → 下降趋势线突破 → 再突破
                              ↑
                          买入点
```

### 13 个买点
- 日线级别买点
- 4 小时级别买点
- 1 小时级别买点
- 任意一个都可

## 仓位管理

### 分批建仓
- 第一批：50% 仓位
- 第二批：回调后再加 50%
- 考虑被锤的情况

### 案例：北方稀土
- 位置：到达前期关键位置
- 资金：今天有资金流入
- 仓位：先买 5 万，跌到 19 块再加 5 万

## 选股要点

### 优选条件
1. 板块热点（有色金属、科技等）
2. 主力资金流入
3. 位置到达支撑位
4. 有技术买点

### 排除条件
1. 石油、黄金、白银（受期货影响大）
2. 大形态边缘的股票（危险）
3. 没有资金流入的股票
4. 今天资金流出的股票

## 案例分析

### 案例：中信特钢
- 位置：到达前期关键位置
- 资金：今天有资金流入
- 形态：60 分钟级别 2B 结构
- 结论：值得备选

### 案例：北方华创
- 位置：到达关键位置
- 资金：今天有资金流入
- 市值：1000 多亿
- 结论：如果跌到 360，可以买入

## 核心结论
1. 等待指数跌到关键区间
2. 资金分成三份，分批建仓
3. 按资金流 + 板块热点选股
4. 结合技术买点（双突破、13 个买点）
5. 考虑被锤情况，设置好加仓位置
