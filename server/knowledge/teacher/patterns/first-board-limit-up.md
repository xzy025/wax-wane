# A股首板涨停模式

## 策略概述
- 首板涨停是云聪老师最喜欢的策略之一
- 大盘好时非常容易赚钱，风险可控
- 大盘不好时会亏得很严重

## 前置条件（三要素）

### 1. 指数情绪
- 涨停股数量：最好 100 只以上
- 跌停股数量：越少越好
- 涨跌比：上涨股票多于下跌股票

### 2. 板块情绪
- 当下热点板块
- 最近的主线（AI、科技、半导体等）
- 板块有持续性

### 3. 赚钱效应
- 最近涨停股表现好
- 没有大量跌停
- 高标股没有连续跌停

## 操作手法

### 手法一：当天排队
- 涨停股排队买入
- 早上看到符合条件的就排队
- 下午可能买到

### 手法二：第二天集合竞价
- 涨幅 1%-5% 最佳
- 超过 5% 要准备被锤
- 主动买套，回调加仓

## 选股标准

### 按资金流筛选
- 当天主力资金流入
- 流入金额越大越好
- 首板看资金流，二板三板不一定看

### 按板块热点筛选
- 当下最热的板块
- 游资最看重的
- 非主线、非热点不做

## 技术形态（13 个买点）

### 1. 大形态突破
- 长期箱体突破
- 长时间三角形突破
- 头肩底突破
- **时间要长，形态要大**

### 2. 茶杯带柄
- 茶杯形态 + 柄部
- 突破瞬间要放量
- 量价配合

### 3. 形态突破回抽确认
- 突破后回抽
- 回抽不破支撑
- 再次上涨

## 仓位管理

### 分批建仓
- 第一批：50% 仓位
- 第二批：回调后再加 50%
- 主动买套，降低均价

### 案例：哈药
- 上周五排队买入
- 下午买到
- 今天涨停赚钱

## 案例分析

### 成功案例：鲁抗
- 条件：资金流入 + 医药板块热点 + 大形态突破
- 操作：上周五排队买入
- 结果：连续涨停

### 成功案例：四川某股
- 条件：资金流入 + 建材板块热点 + 形态突破回抽确认
- 操作：排队买入
- 结果：吃了 3 个涨停

### 成功案例：云南楚业
- 条件：大形态突破 + 茶杯带柄结构
- 操作：排队买入
- 结果：一波大涨

### 失败案例：豆神教育
- 条件：资金流入 + 教育板块热点 + AI 结合
- 操作：排队买入
- 问题：盘后公布财务造假利空
- 教训：A 股基本面分析可能有造假

## 游资思维

### 游资最看重的
1. 当天板块热点
2. 当下主线
3. 潮点（市场情绪）

### 游资不太看重的
- 技术形态
- 基本面
- 财务数据

### 结合方法
- 资金流 + 板块热点 + 技术形态
- 或只看游资最热的板块

## 风险提示

### 大盘不好时
- 涨停股少，跌停股多
- 高标股连续跌停
- 不适合重仓做首板

### 可以轻仓练习
- 大盘不好时可以轻仓
- 练习手感
- 但绝对不能重仓

## 核心结论
1. 指数情绪、板块情绪、赚钱效应三要素
2. 按资金流 + 板块热点选股
3. 技术形态：大形态突破、茶杯带柄、突破回抽确认
4. 分批建仓，主动买套
5. 大盘好时重仓，大盘不好时轻仓或不做
