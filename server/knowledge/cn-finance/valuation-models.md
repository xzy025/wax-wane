# 估值模型

## 概述

提供估值模型的计算框架，用户自行填入假设参数。支持绝对值估值（DCF、DDM）和相对估值（可比公司、可比交易）两类方法。**只输出理论估值区间，不输出目标价或买卖建议。**

## DCF 模型完整框架

### 第一步：计算自由现金流

**FCFF（企业自由现金流）**：
```
FCFF = EBIT × (1 - t) + 折旧与摊销 - 资本支出 - 营运资本增加
```

**FCFE（股权自由现金流）**：
```
FCFE = FCFF - 利息费用 × (1 - t) + 净借款增加
     = 净利润 + 折旧与摊销 - 资本支出 - 营运资本增加 + 净借款增加
```

### 第二步：确定预测期
- **显式预测期**：通常 5-10 年（周期性行业取完整周期）
- **终值期**：显式预测期之后的所有年份
- 预测期越长，终值占比越小，模型越稳健

### 第三步：估算 WACC

```
WACC = (E/V) × Re + (D/V) × Rd × (1 - t)
其中：
  E = 股权市值，D = 债务市值，V = E + D
  Re = 股权成本 = Rf + β × ERP
  Rd = 债务成本（公司加权平均借款利率）
  t = 实际所得税率
```

**中国市场参数参考范围** [待验证]：
| 参数 | 典型范围 | 获取方式 |
|------|---------|---------|
| Rf（无风险利率） | 2.5-3.5%（基于 10Y 国债） | 中国债券信息网 |
| ERP（股权风险溢价） | 5-8% | Damodaran 数据库 / 本地研究 |
| β（贝塔系数） | 0.6-1.8 | 对沪深 300 做 2-5 年周收益率回归 |
| β 去杠杆公式 | β_unlevered = β_levered / (1 + (1-t) × D/E) | 用于可比公司 β 调整 |
| Rd（税前债务成本） | 3-6%（AAA）、6-10%（AA）、10%+（A及以下） | 公司年报 / 债券 YTM |
| 最优资本结构 | 用可比公司中位数 D/E 或公司目标资本结构 | — |

### 第四步：终值计算（两种方法比较）

| 方法 | 公式 | 优点 | 缺点 |
|------|------|------|------|
| 永续增长法 | TV = FCF_n × (1+g) / (WACC - g) | 理论上最正确 | 对 g 高度敏感 |
| 退出倍数法 | TV = EBITDA_n × 退出倍数 | 市场参照性强 | 倍数选择有主观性 |

**中国适用性**：
- 永续增长率 g 通常取 2-3%（接近长期 GDP 增速/CPI）
- 退出倍数取行业 EV-EBITDA 中位数的 50-70%（保守原则）
- **建议两种方法都算，TV 取两者较低值或加权平均**

### 第五步：折现与股权价值

```
企业价值 EV = Σ(FCFF_t / (1+WACC)^t) + TV / (1+WACC)^n
股权价值 = EV - 债务市值 - 少数股东权益 - 优先股 + 现金及等价物
每股价值 = 股权价值 / 总股本（稀释后）
```

### 敏感性分析（标准做法）
对以下变量做二维敏感性：
- WACC ±0.5% ~ ±2%（间隔 0.5%）
- g ±0.25% ~ ±1%（间隔 0.25%）
- 收入增速 ±1% ~ ±5%（间隔 1%）

## DDM 模型框架

```
每股价值 = Σ(DPS_t / (1+r)^t) + TV / (1+r)^n
其中 TV = DPS_{n+1} / (r - g)
```

### 中国 A 股适用性评估
| 适用场景 | 不适用场景 |
|---------|----------|
| 银行股（分红稳定） | 高增长不分红的成长股 |
| 公用事业（分红率高） | 亏损公司 |
| 成熟消费龙头 | 分红政策频繁变动的公司 |

### 股利折现模型变体
- **Gordon 增长模型**：DPS × (1+g) / (r - g) — 适用于稳定增长期
- **两阶段模型**：高增长阶段 + 稳定阶段 — 适用于中国多数公司
- **H 模型**：增长率线性衰减 — 适用于行业从成长走向成熟的公司

## 可比公司估值

### 完整筛选流程
1. 确定目标公司所属申万三级/四级行业
2. 筛选同行业 + 市值 0.3x-3x + 上市 > 1 年 + 非 ST
3. 检查业务可比性（主营构成余弦相似度 > 0.7）
4. 如样本不足 5 家：放宽至申万二级行业
5. 如样本仍不足：改用全行业估值分位法

### 估值倍数详解
| 倍数 | 计算 | 考虑因素 |
|------|------|---------|
| PE (TTM) | 股价 / 近 12 个月 EPS | 用归母扣非 EPS 更可靠 |
| PE (Forward) | 股价 / 预测 EPS | 需一致预期数据（Wind/同花顺） |
| PB | 股价 / 每股净资产 | 注意商誉和无形资产对净资产的扭曲 |
| PS (TTM) | 总市值 / 近 12 个月收入 | 收入不易被操纵，适合扭亏期公司 |
| EV-EBITDA | (市值+净债务) / EBITDA | 排除资本结构和折旧政策差异 |
| PEG | PE / 净利润增速(%) | >1 偏贵、<0.5 偏便宜（经验规则） |

### 分行业估值倍数选择（完整版）
| 申万一级 | 首选 | 次选 | 参考 | 不适用 | 原因 |
|---------|------|------|------|--------|------|
| 银行 | PB | PE | P/PPOP | EV-EBITDA | 高杠杆经营 |
| 非银-券商 | PB | PE | P/BV | EV-EBITDA | 强周期+高杠杆 |
| 非银-保险 | P/EV | PB | PE | — | 内含价值是保险核心 |
| 食品饮料 | PE | PEG | EV-EBITDA | PB | 稳定盈利 |
| 医药生物 | PE | PEG | PS(亏损期) | PB | 研发开支扭曲净资产 |
| 电子 | PE | EV-EBITDA | PS | PB | CAPEX 大，折旧影响 |
| 计算机 | PS | PE | EV/Revenue | PB | 轻资产，利润波动大 |
| 电力设备（新能源）| PE | PEG | EV-EBITDA | PB | 高增长+CAPEX 大 |
| 房地产 | PB(NAV) | PE | P/RNAV | EV-EBITDA | 资产价值是核心 |
| 建筑装饰 | PE | PB | EV-EBITDA | — | 订单驱动 |
| 汽车 | PE | EV-EBITDA | PS | PB | CAPEX 大 |
| 公用事业 | PE | PB | EV-EBITDA | — | 现金流稳定 |
| 化工 | PE | EV-EBITDA | PB | — | 周期性强 |

## 分析流程

1. 确认分析目标和方法（DCF/DDM/comps/综合）
2. 获取公司和可比公司财务数据
3. **用户填入关键假设参数**（增长率、WACC、终值假设等）
4. 运行模型计算并做敏感性分析
5. 输出**理论估值区间**（不是目标价）
6. 交叉检验：至少两种方法交叉验证
