# 财务报表分析

> **数据源**：本 skill 使用用户配置的数据接口（AKShare / Tushare）获取财务报表数据。
> 用户须自行注册数据源账号并配置 API token。Token 存储在用户本地，不会上传或共享。

## 概述

分析中国上市公司财务报表，支持 CAS（中国会计准则）下的三表联动分析、杜邦分析（含五步分解）、同比环比分析、盈利质量评估。

## CAS vs IFRS 差异完整对照

### 逐科目差异表
| 会计领域 | CAS 处理 | IFRS 处理 | 影响分析 |
|---------|---------|----------|---------|
| 存货计价 | 先进先出法、加权平均法、个别计价法 | 禁止后进先出法，其他同 | CAS 与 IFRS 趋同 |
| 长期资产减值 | **禁止转回**（固定资产、无形资产、商誉） | 允许转回（商誉除外） | CAS 下利润更保守 |
| 存货减值 | 允许转回 | 允许转回 | 一致 |
| 政府补助 | 与资产相关的确认为递延收益，分期转入损益 | IAS 20：可选择递延法或直接抵扣法 | CAS 更倾向递延 |
| 合并报表控制 | 权力+可变回报+权力影响回报的能力（三要素） | 同 IFRS 10 三要素模型 | 趋同 |
| 关联方认定 | 同受国家控制的企业之间属于关联方 | 同受政府控制不一定构成关联方 | CAS 关联方范围更广 |
| 开发支出资本化 | 五个条件（同 IFRS） | 六个条件（多一个"技术可行性"明确要求） | 实质趋同 |
| 投资性房地产 | 后续计量可用成本模式或公允价值模式 | 同 | 一致 |
| 收入确认 | 五步法（2017 新收入准则） | IFRS 15 五步法 | **完全趋同** |
| 租赁 | 单一使用权资产模型（2018 新租赁准则） | IFRS 16 单一模型 | **完全趋同** |

### A股特有科目详解
| 科目 | 位置 | 含义 | 分析要点 |
|------|------|------|---------|
| 递延收益 | 负债（流动/非流动） | 与资产相关或与收益相关的政府补助 | 未来将转入损益，增加利润但不产生现金流 |
| 专项储备 | 所有者权益 | 高危行业计提的安全生产费、维简费 | 计提减少利润，实际使用时不减利润 |
| 少数股东损益 | 利润表内单独列示 | 归属于少数股东的净利润 | 归母在净利润下方单独列示，"归母净利润"才是核心 |
| 其他权益工具 | 所有者权益 | 优先股、永续债等 | 明股实债的常见藏身之处 |
| 合同负债 | 流动负债 | 预收款项（新收入准则改名） | 高合同负债 = 预收款多 = 客户话语权强 |

## 三表结构详解

### 资产负债表（BS）
```
资产 = 负债 + 所有者权益
流动资产 + 非流动资产 = 流动负债 + 非流动负债 + 归属于母公司的所有者权益 + 少数股东权益
```
| 关键勾稽 | 公式 |
|---------|------|
| 期末未分配利润 | 期初未分配利润 + 本期归母净利润 - 本期分红 |
| 期末盈余公积 | 期初盈余公积 + 本期净利润 × 10%（法定） |

### 利润表（IS）
```
营业收入
- 营业成本
= 毛利
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 其他收益（政府补助等）
+ 投资收益
+ 公允价值变动收益
- 信用减值损失
- 资产减值损失
= 营业利润
+ 营业外收入 - 营业外支出
= 利润总额
- 所得税费用
= 净利润
  其中：归母净利润
  其中：少数股东损益
```

### 现金流量表（CF）
| 分类 | 关键流入 | 关键流出 | 分析关注点 |
|------|---------|---------|----------|
| 经营活动 | 销售商品收到的现金 | 购买商品支付的现金、支付给职工的现金 | **经营现金流 > 净利润 → 盈利质量高** |
| 投资活动 | 收回投资、取得投资收益 | 购建固定资产、对外投资 | CAPEX 需与折旧摊销匹配 |
| 筹资活动 | 吸收投资、借款 | 偿还债务、分红 | 关注债务结构和融资成本 |

### 三表勾稽核心
1. **IS → BS**：净利润（扣除分红）→ 未分配利润
2. **IS → CF**：净利润 → 调整非现金项目 → 经营活动现金流（间接法）
3. **BS → CF**：资产/负债的期初期末变动 → 现金流各项目
4. **审计关键**：CF 期末现金 = BS 货币资金期初期末差

## 完整财务比率体系

### 盈利能力（含五步杜邦分解）
| 指标 | 公式 | 中国经济含义 |
|------|------|------------|
| ROE（归母） | 归母净利润 / 平均归母净资产 | 股东回报率核心指标 |
| ROA | 净利润 / 平均总资产 | 资产运用效率 |
| 毛利率 | (收入 - 成本) / 收入 | 产品定价权和成本控制 |
| 净利率 | 净利润 / 收入 | 综合盈利能力 |
| 扣非净利率 | 扣非净利润 / 收入 | 剔除一次性项目后的持续盈利 |
| EBITDA Margin | EBITDA / 收入 | 不同资本结构公司的可比盈利 |

### 偿债能力
| 指标 | 公式 | 合理区间参考 |
|------|------|------------|
| 流动比率 | 流动资产 / 流动负债 | 1.5-2.5（制造业），1.0-1.5（轻资产） |
| 速动比率 | (流动资产 - 存货) / 流动负债 | 1.0-1.5 |
| 资产负债率 | 总负债 / 总资产 | 40-60%（制造业），>80%（银行，正常） |
| 利息保障倍数 | EBIT / 利息费用 | >3（安全），<1（危险） |
| 有息负债率 | 有息负债 / 总资产 | 越低越好，>50% 需关注 |
| 净负债率 | (有息负债 - 现金) / 净资产 | 现金充沛的公司可能为负 |

### 营运能力
| 指标 | 公式 | 行业差异 |
|------|------|---------|
| 应收账款周转天数 | 365 × 平均应收 / 收入 | 消费品<30天，工程类>180天 |
| 存货周转天数 | 365 × 平均存货 / 成本 | 快消<60天，白酒>300天正常 |
| 应付账款周转天数 | 365 × 平均应付 / 成本 | 越大 → 占用上游资金越多 |
| 现金转换周期 | 应收天数 + 存货天数 - 应付天数 | 负值 → 用别人的钱运营 |

### 成长能力
| 指标 | 公式 | 分析注意事项 |
|------|------|------------|
| 收入 CAGR | (本期收入/基期收入)^(1/n)-1 | 需看 3-5 年，单年可能受基数影响 |
| 归母净利润 CAGR | 同上 | 需与收入增速匹配，大幅背离需解释 |
| 经营现金流增速 | 同上 | 利润增长但现金流不增长 → 盈利质量下降 |

## 五步杜邦分解

```
ROE = (净利润/EBT) × (EBT/EBIT) × (EBIT/收入) × (收入/总资产) × (总资产/净资产)
    = 税负因子 × 利息负担因子 × 经营利润率 × 总资产周转率 × 权益乘数
```

| 因子 | 含义 | 恶化信号 |
|------|------|---------|
| 税负因子 | 实际税率的影响 | 税率突然大幅变化 |
| 利息负担因子 | 财务杠杆的正面效应（借债→息税前利润更多归股东） | < 0.5 → 利息负担过重 |
| 经营利润率 | 核心经营盈利能力 | 持续下降 → 竞争恶化 |
| 总资产周转率 | 资产运营效率 | 下降而收入增长 → 资产扩张快于收入 |
| 权益乘数 | 财务杠杆 | 快速上升 → 加杠杆 |

## 盈利质量分析

### 现金流匹配度
- 经营现金流 / 净利润 > 1.0 → 高质量
- 0.5-1.0 → 中等质量，需关注原因
- < 0.5 持续 → 盈利质量差，利润可能无法转化为现金
- 为负 → 严重问题（除金融/地产等特殊行业）

### 非经常性损益陷阱
- 扣非净利润 / 归母净利润 < 0.7 → 利润依赖一次性项目
- 政府补助占净利润比 > 30% → 持续盈利能力存疑
- 公允价值变动收益波动大 → 利润不稳定

## 分析流程

1. 获取最近 5 年（或 10 个报告期）的财务报表数据
2. 计算核心财务比率并标注行业分位
3. 五步杜邦分析拆解 ROE 变化驱动因素
4. 经营现金流 vs 净利润匹配度分析
5. 识别非经常性损益和异常科目变动
6. 输出结构化分析报告

**财务造假三大手法与识别方法**

根据证监会2024年执法数据，虚构业务型造假占全部财务造假案件的88%，其次为滥用会计政策型和并购子公司隐瞒型。

**虚构业务型**（"合同流+资金流+发票流"三流合一但无真实商业活动）：
- 预警信号：应收账款/预付款增速持续远超营收；与少数客户或关联方高度集中；回款周期异常长；毛利率在行业整体波动时却异常稳定

**滥用会计政策型**（提前确认收入、跨期确认、资产减值计提不足）：
- 预警信号：各季度收入呈"前低后高"规律性模式；商誉占总资产比例持续攀升；资产减值损失突然大幅计提（集中出现在年末）

**并购子公司隐瞒型**（子公司向母公司隐瞒造假，影响合并报表）：
- 预警信号：合并报表与母公司报表差距明显扩大；并购后业绩承诺期恰好完成但随后骤降；子公司现金流显著弱于其利润表现

### 2025年执法新动向（截至2026年5月）

**执法升级三大里程碑**

1. **全链条追责破冰（2025年6月）**：证监会首次对配合财务造假的非中介机构第三方个人直接处罚——越博动力案中，配合造假的于某、贺某分别被罚款200万、30万元。此前监管仅能处罚上市公司及董监高，"帮凶"无法被直接追责。
2. **退市提速常态化**：2025年因财务造假触及重大违法强制退市的公司已达9家，同比增长约30%。退市不再是终点——锦州港已退市但副总裁仍于2025年7月因涉嫌违规披露被逮捕，证明"退市后仍追责"已成现实。
3. **裁量规则统一（2025年3月施行）**：《中国证监会行政处罚裁量基本规则》正式施行，首次明确共同违法"先整体认定后区分处罚"的裁量规则，处罚标准跨派出机构统一。

- 年度审计机构多次更换，或审计意见前后矛盾
- 公司退市整理期前大量内部人突击减持
