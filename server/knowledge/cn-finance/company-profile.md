# A股公司一页纸速览

> **数据源**：本 skill 依赖 financial-analysis 核心层的 AKShare/Tushare connector 获取公司信息、行情和财务数据。

## 概述

生成 A 股上市公司的标准化一页纸速览报告，覆盖公司概况、股权结构、核心财务、估值水平和行业地位。

## 速览模板结构

### 1. 公司概况
- 公司全称和简称
- 股票代码（含交易所后缀：.SH / .SZ / .BJ）
- 上市日期
- 所属行业（申万一级/二级/三级）
- 注册地/办公地（省份城市）
- 实际控制人/控股股东
- 实际控制人性质（央企/地方国企/民企/外资/无实际控制人）
- 主营业务描述（一句话+主营构成 Top3）
- 员工人数

### 2. 股权结构
- 控股股东及持股比例
- 实际控制人及其控制链条
- 前十大股东及其性质（国有法人/民营/外资/机构/个人）
- 前十大股东持股合计
- 控股股东股权质押比例（> 80% 红色预警）
- 近半年是否有减持计划公告/大宗交易减持
- 是否有回购计划及进度
- 股权激励/员工持股情况

### 3. 核心财务（近 3-5 年）
- 营业收入及同比增速（3 年 CAGR）
- 归母净利润及同比增速
- 扣非净利润（与归母净利润对比，看非经常性占比）
- ROE（加权）（杜邦拆解：净利率×周转率×杠杆倍数）
- ROA
- 毛利率 / 净利率（趋势：扩张/收窄/平稳）
- 期间费用率（销售+管理+研发+财务）/ 收入
- 资产负债率
- 经营现金流净额 / 净利润（盈利质量）
- 自由现金流（经营现金流 - CAPEX）
- 分红率（近 3 年）

### 4. 估值水平
- PE (TTM) — 扣非口径优先
- PE 近 5 年历史分位
- PB 及近 5 年历史分位
- PS (TTM)
- EV-EBITDA
- 总市值 / 流通市值
- 近 1 月/3 月/1 年涨跌幅
- 近 3 年年化波动率 / 最大回撤

### 5. 行业地位
- 申万三级/四级行业中的市值排名
- 收入在行业中的排名
- 核心产品市场份额估算（如有公开数据）
- 主要竞争对手（2-3 家，含代码）
- 护城河定性评估（品牌/切换成本/网络效应/成本优势/规模优势/牌照壁垒）

### 6. 公司治理快评
| 维度 | 评估 |
|------|------|
| 股权结构稳定性 | 实控人是否明确、近 3 年是否变更 |
| 管理层稳定性 | 董事长/总经理任期、近 2 年是否变更 |
| 信息披露质量 | 交易所信息披露评级（A/B/C/D） |
| 审计意见 | 近 3 年审计意见类型 |
| 分红记录 | 连续分红年数、分红率稳定性 |
| 关联交易 | 关联交易/收入比例 |
| 对外担保 | 对外担保/净资产比例 |

### 7. 关键风险提示
- 大股东高比例质押（> 80%）
- 商誉/净资产 > 30%
- 应收账款/收入 > 50% 且持续上升
- 客户/供应商集中度 > 50%
- 证监会/交易所处罚/问询记录
- 重大诉讼/仲裁
- 行业政策重大不确定性

## A股公司实际控制人分类体系

| 实控人类型 | 二级分类 | 特征 | 估值影响 |
|-----------|---------|------|---------|
| 央企 | 国务院国资委直属 / 央企集团控股 | 治理规范、融资便利、市值管理考核 | 通常有国企溢价（确定性强） |
| 地方国企 | 省属 / 市属 / 区县属 | 分化大：优质（茅台/五粮液）vs 一般 | 视经营效率和地方财力而定 |
| 民营企业 | 创始人控股 / 家族控股 / 管理层控股 | 效率高但治理风险分化大 | 治理好的民企可享受成长溢价 |
| 外资企业 | 外资控股 / 中外合资 | 通常在制造业和消费品 | — |
| 无实际控制人 | 股权分散 | 管理层控制，可能有控制权争夺风险 | 通常有折价 |

## 关键风险指标定量阈值

| 风险指标 | 黄灯阈值 | 红灯阈值 | 说明 |
|---------|---------|---------|------|
| 控股股东质押比例 | 50-80% | > 80% | 平仓风险+掏空动机 |
| 商誉/净资产 | 20-30% | > 30% | 减值风险 |
| 应收账款/营业收入 | 30-50% | > 50% | 回款恶化 |
| 关联交易/收入 | 20-30% | > 30% | 定价公允性风险 |
| 对外担保/净资产 | 30-50% | > 50% | 或有负债风险 |
| 前5大客户集中度 | 50-70% | > 70% | 单一客户依赖 |
| 存货/总资产 | 20-30% | > 30% | 存货跌价风险 |
| 短期借款/货币资金 | 50-100% | > 100% | 流动性风险 |

## A股公司业务分类

### 主营构成分析
- 获取最近年报/中报的分产品/分行业/分地区收入构成
- 判断核心业务、成长业务和边缘业务
- 识别收入确认政策（总额法 vs 净额法）

### 业务模式识别
| 模式 | 典型特征 | 关注指标 |
|------|---------|---------|
| 产品型 | 制造/销售有形产品 | 毛利率、产能利用率、存货周转 |
| 项目型 | 按合同/项目交付 | 在手订单、应收账款、履约进度 |
| 平台型 | 双边/多边网络 | GMV、take rate、用户数/活跃度 |
| 订阅/SaaS型 | 年/月经常性收入 | ARR/MRR、续费率/流失率(churn) |
| 许可/IP型 | 技术许可/品牌授权 | 授权收入占比、合同期限 |

## 分析流程

1. 输入股票代码，获取公司基本信息（代码、名称、行业、上市日期、注册地）
2. 获取股东信息（控股股东、前十大、质押比例、减持动态）
3. 获取财务数据（近 3-5 年核心指标，含季度）
4. 计算关键比率和趋势
5. 获取行情数据（当前估值、涨跌幅、历史分位）
6. 获取行业排名（在同申万三级行业中排位）
7. 扫描关键风险指标
8. 按模板格式化输出
