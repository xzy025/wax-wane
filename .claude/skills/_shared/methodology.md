## 一、交易模式接入（/ta 与 /equity 用）

8 个交易模式定义在 `src/agent/tradingPatterns.ts` 的 `TRADING_PATTERNS`，**分析前读取该文件**取最新内容：

| id | 名称 | 类别 |
|---|---|---|
| `2b-buy` | 2B买入模型 | 云聪交易模式 |
| `strong-dip` | 强势股低吸 | 云聪交易模式 |
| `first-board` | 首板涨停 | 云聪交易模式 |
| `morning-dip` | 早盘指数低吸 | 云聪交易模式 |
| `wyckoff` | Wyckoff量价理论 | 理论框架 |
| `dow` | 道氏理论 | 理论框架 |
| `price-action` | Al Brooks价格行为 | 理论框架 |
| `sentiment` | A股情绪周期 | 理论框架 |

- 用户在参数里给了模式 id（可多个）→ 取每个模式的 `analysisGuide`，**逐条对照其关键要素**分析当前标的，明确指出「符合 / 不符合」。
- 用户**没给** → `/ta` 默认套用技术类理论框架 `wyckoff` + `dow` + `price-action`；`/equity` 在此基础上加 `sentiment`。
- **Larry Williams 短线交易秘诀始终叠加**（见「二·补充」），不论是否选了其他模式——它是 `/ta`、`/equity` 技术面的固定组成部分。
- 多个模式有交叉时，指出彼此联系（对齐 `buildPatternContext` 的综合要求）。

## 二、技术面映射（对齐 technical.agent.ts）

取约 60 根日线，以最后一根为「最新」：

- **涨跌幅**：5 日、20 日收盘涨跌 %。
- **20 日区间位置**：`(收盘 - 20日最低) / (20日最高 - 20日最低) * 100`，0%=最低、100%=最高。
- **量能**：`5日均量 / 20日均量` = 量比。`>1.5` 放量 / `<0.7` 缩量 / 其余 平量。
- **K 线形态**（用最新一根）：
  - 锤子线（看涨）：下影 > 实体×2 且 上影 < 实体×0.5
  - 射击之星（看跌）：上影 > 实体×2 且 下影 < 实体×0.5
  - 十字星（犹豫）：实体 < (高-低)×0.1
- **Wyckoff 阶段**（20 日趋势 × 量比）：
  - 上升 + 量比>1.2 → 上涨期（放量上涨，健康）
  - 上升 + 量比<0.8 → 上涨末期（缩量上涨，注意见顶）
  - 下降 + 量比>1.2 → 派发期（放量下跌，主力出货）
  - 下降 + 量比<0.8 → 下跌末期/吸筹期（缩量下跌，可能见底）
  - 其余 → 横盘整理
- **支撑/阻力**：20 日最低 / 20 日最高。
- **技术信号**：多头分 = 锤子线 + 上升趋势 + (量比>1.2且上升)；空头分 = 射击之星 + 下降趋势 + (量比>1.2且下降)。多头 > 空头+1 → **偏多**；空头 > 多头+1 → **偏空**；否则 **中性**。

## 二·补充、Larry Williams 短线交易秘诀（拉里·威廉斯，技术面固定叠加）

出自《短线交易秘诀》(Long-Term Secrets to Short-Term Trading)。基于日 OHLCV 可计算的核心工具，全部在 `/ta`、`/equity` 技术面里给出：

- **Williams %R（10日）**：`%R = (10日最高HH − 收盘) / (HH − 10日最低LL) × (−100)`，取值 −100~0。
  - `> −20` 超买；`< −80` 超卖；`−20~−80` 中性。
  - **结合趋势用**：上升趋势中 %R 跌入超卖区＝顺势回调买点；下降趋势中超买不轻易抄底，反弹是减仓位。
- **波动突破入场（Volatility Breakout）**：以前一日波幅 `R = 昨日(高−低)` 为基准（k=0.5）：
  - 多头触发位 ≈ `参考价 + 0.5×R`，空头触发位 ≈ `参考价 − 0.5×R`（参考价取最新收盘，作次日开盘近似）。
  - 报告里给出当前的「上破转强位 / 下破转弱位」。
- **Oops! 反转**（用最新一根的开盘 vs 前一日高低）：
  - 跳空**低开跌破昨日最低**、盘中又收回其上 → **看涨 Oops（买）**；
  - 跳空**高开越过昨日最高**、又跌回其下 → **看跌 Oops（卖）**；不满足则注明「无 Oops」。
- **Smash Day（突袭日）反转**：
  - 收盘**跌破前一日最低**＝下跌突袭日 → 次日若上破突袭日最高＝反转买点；
  - 收盘**升破前一日最高**＝上涨突袭日 → 次日若下破其最低＝反转卖点；给出最新是否构成突袭日及触发位。
- **短线摆动点（Ringed High/Low）**：某日高点高于左右相邻两日＝短线高点；低点低于左右相邻两日＝短线低点。标出**最近一个短线高点/低点**作为短线阻力/支撑与结构参考。
- **TDW/TDM 季节性**（可选，A股样本有限，仅作定性提示）：星期几 / 月内交易日的统计偏向。

**输出**：技术面段落新增一行块「**Larry Williams 短线信号**」，含 %R 读数与超买/超卖、上破/下破触发位、Oops/Smash 判定、最近短线摆动高低点，并给出短线倾向（偏多/中性/偏空）。

## 三、基本面分级（对齐 fundamental.agent.ts）

- **PE**：<15 低估 / <30 合理 / <60 偏高 / ≥60 高估
- **PB**：<1 破净(可能低估) / <3 合理 / <5 偏高 / ≥5 高估
- **ROE**：>20% 优秀 / >15% 良好 / >10% 一般 / ≤10% 偏低
- **市值**：以「亿」展示（值 / 1e8）。
- **估值综合**：
  - PE<30 且 ROE>15 → 估值合理，盈利能力强
  - PE<15 且 ROE>10 → 低估，有投资价值
  - PE>60 或 (PB>5 且 ROE<10) → 估值偏高，注意风险
  - 其余 → 估值中性
- 若拿到归档深度报告，附其摘要并标注生成日期 / 是否 stale（>30 天）。

## 四、消息面情绪（对齐 stock-news.agent.ts）

- 取最多 10 条，展示 **Top 5 标题**（含时间，snippet 截断 ~100 字）。
- 关键词计数（命中即 +1，按词表去重）：
  - 利好词：`利好 增长 突破 创新高 涨停 大涨 盈利 超预期 增持 回购`
  - 利空词：`利空 下跌 暴跌 亏损 减持 违规 处罚 风险 警示 退市`
- 情绪判定：利好 > 利空+2 → 偏正面 / 利好 > 利空 → 略偏正面 / 利空 > 利好+2 → 偏负面 / 利空 > 利好 → 略偏负面 / 否则 中性。

## 五、综合判断（/equity 用，对齐 synthesizer.agent.ts）

汇总技术/基本/消息三维度的方向信号：

- 多头信号词：上涨/上升/吸筹/修复/低估/偏正面/偏多
- 空头信号词：下跌/下降/派发/退潮/高估/偏负面/偏空
- 多 > 空 → **综合偏多**；空 > 多 → **综合偏空**；持平 → **中性，建议观望**。
- 列出各维度结论，指出分歧点，并结合所选交易模式给出可执行建议（入场/仓位/止损位参考交易模式的 analysisGuide）。

## 六、报告格式

`/ta`、`/fa` 只输出对应单段。`/equity` 输出完整四段（对齐 orchestrator `buildReport`）：

```
## 📊 个股分析报告 — {名称} — {YYYY/M/D}
### 一、技术面分析
...
### 二、基本面分析
...
### 三、消息面分析
...
### 四、综合评价
...
```

每段末尾用粗体给该维度信号（如 `技术信号: **偏多**`）。结尾统一附一行免责声明：
`> 本分析由数据自动生成，仅供研究参考，不构成投资建议。`
