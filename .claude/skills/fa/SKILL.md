# /fa — 标的基本面分析

对单只 A 股做**基本面**分析（不含技术面/消息面）。

## 输入

`/fa <6位代码或中文名>`
- 例：`/fa 600519`、`/fa 贵州茅台`

## 步骤

1. **解析标的**：按 `.claude/skills/_shared/data-sources.md` 第 0 节规范化为 6 位代码。
2. **取基本面快照**：按 data-sources.md 第 2 节取 PE/PB/ROE/市值/行业/EPS 等（项目服务在跑优先，否则东财直取，再否则 akshare 财报自算）。
3. **取归档深度报告**：若项目服务在跑，按第 4 节取最新归档报告；有则附摘要 + 生成日期 + 是否 stale。
4. **分析**：按 `.claude/skills/_shared/methodology.md` 第三节做估值/盈利分级与综合估值判断。
5. **输出**：只出基本面单段，末尾粗体给 `估值判断`，附免责声明。

```
## 🏢 基本面分析 — {名称}（{代码}） — {YYYY/M/D}
- 行业 / 总市值
- 市盈率(PE)（分级）
- 市净率(PB)（分级）
- 净资产收益率(ROE)（分级）
- 每股收益(EPS)
- 估值判断: **{合理/低估/偏高...}**
- （可选）历史深度报告摘要：{...}（生成于 {日期}，{是否过时}）

> 本分析由数据自动生成，仅供研究参考，不构成投资建议。
```

## 注意
- 只读项目源码，不修改 `src/`、`server/`。
- 无需任何 API key；不使用 tushare。
